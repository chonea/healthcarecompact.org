How to install map on a website

Adding the customized map to your website by means of using our tool is simple.
The only thing you need to do is to upload several files on your server and insert our HTML code in any desirable place of your web page.
Most of the content management systems and blog engines allow inserting HTML code and are compatible with our tool.

1. Unziping the files

Unzip the file xxxxx.zip be means of WinZip, 7-Zip, or the built-in Microsoft Windows extraction software.

2. Testing the map

  • Open the folder with unzipped files and click on the file locator_start.html;
  • You will see the map in your browser;
  • Go to the next step to customize the XML file;
  • You can easily see any changes you made by refreshing the page.
* You can test the map on your PC or your server. Sometimes, the links may not work locally, because of the Adobe security settings. You can easily fix it by clicking on: Configuring Adobe Flash Player Settings.

3. Customizing the XML file

All adjustments and customizations should be made in the XML file. We advise you to use free Notepad++. Certainly, you can use any other HTML and plaintext editors as well.

The configurations and customizations can be made on your local computer. After customization you have to upload the edited files on your web server.

General settings
  name_color="FFFFFF" Color of name of State
  name_shadow_color="000000" Color of shadow of a name
  name_font_size = "11" Font size
  background_color = "FFFFFF" Color of background. Use 0 to get a transparent background.
  borders_color = "FFFFFF" Color of borders
  abbrevation_color = "FFFFFF" Color of abbrevations (only for map of USA)
  abbrevation_color_hover = "DCDCDC" Rollover color of abbrevations (only for map of USA)
  popup_background_color = "FFFFFF" Color of background of popup box
  popup_borders_color = "999999"   Color of borders of popup box
  popup_text_color = "000000" Color of text in popup box
  popup_text_size = "10" Font size of text in popup box
  sounds = "on" On/off sounds
  animated_state = "off" On/off rollover effect
  gradient = "off" On/off gradient on States

For customizing colors, you should use 6-digits color code. For your convenience, you may use Color picker provided with this map.

For each State you can change the name, color, rollover color, link URL, the description, the image, and the target frame.

Settings for each State
<id> Number of State in the list in order. Do not change the order of the states.
<name> Here you can set the name that will appear when a State is hovered over
<link> The URL a click will send the user to. Must be in the format:
http://www.yourdomain.com or
http://www.yourdomain.com/page.html or /directory/page.html
It is also possible to define custom JavaScript.
<comment> The text that will be displayed inside of a popup box for the specified State. To begin a new line, press "Enter", or use a code ||
<color_map> Color of the State. For example dark blue - 0x0000FF.
<color_map_over> Color of the State on hover.
<frame> You can define the target frame for link. It is used, if it is necessary for you to open a new window of a browser, or to display the data in the frame, without reloading pages.
<image> Name of image file. It can be the image in the size 64*64 px (jpg). This image will be displayed in the popup-window.

Sample XML code for one state
<state>
  <id>1</id>
  <name>Alabama</name>
  <link>http://www.google.com</link>
  <comment>Text comment</comment>
  <color_map>0x7798BA</color_map>
  <color_map_over>0x366CA3</color_map_over>
  <frame></frame>
  <image>photo.jpg</image>
</state>


4. Uploading the files to your server

Upload the files to the same folder where your web page is. You can also create a folder in your server and name it as you like and upload those files into this newly made folder.

It is needed to upload the following files:
  • locator_map.swf - the map-engine file
  • data.swf - the map-data file
  • locator_map.xml - the file with settings and data
  • swfobject.js - is a popular SWFObject script. SWFObject is an easy-to-use and standards-friendly method to embed Flash content.
Do not change the name of the files!

5. Embedding the Flash object into your website

Copy and paste the following HTML code into your web page in any place you want.

The following HTML code needs to be used if the files are located in the same catalog with your web page where you plan to show the map.

If the files are located in any other catalog, please specify the path to locator_map.swf, locator_map.xml, data.swf and swfobject.js in our HTML code to the folder in which you uploaded the map files.

This is an example of the HTML code with the path: 

<script type="text/javascript" src="/folder/swfobject.js"></script>
<script type="text/javascript">
        var params = {
          wmode: "transparent",
          menu: "false"
        };
        var flashvars = {
          datafile: "/folder/data.swf",
          xmlfile1: "/folder/locator_map.xml"
        };
        swfobject.embedSWF("/folder/locator_map.swf", "locator_map", "500", "400", "9.0.0", "", flashvars, params);
</script>
<div style="width:500px;height:400px;">
 <div id="locator_map">
  <p>Alternative content</p>
 </div>
</div>

You can customize any size of the map in the HTML code by means of changing the width and height in the red parts of the code.

Troubleshooting.

Q. Javascript or linking is not working properly when I test my map locally?

A. This problem is apparently caused by your local Flash security settings. The problem will disappear if you upload the map on your server. You can easily fix this problem by changing your local security setting.

Please read the next paragraph for instructions how to configure the Adobe Flash Player Security Settings.

Configuring Adobe Flash Player Security Settings

Some little changes in Adobe Flash Player and above settings may be required for your Flash map to be function properly on your local PC.

Please follow the instructions if your map does not function properly:

1. Open the Adobe Flash Player Setting Panel
http://www.macromedia.com/support/documentation/en/flashplayer/help/settings_manager04.html
2. Click on "Edit locations..." dropdown box under the Adobe Flash Player's setting panel.
3. Click on "Add location".
4. Select the folder (or the disc C or D) that contains of the map.
5. Refresh the web page with your map.

Your map should work properly in your PC if you followed those steps correctly.
